# VEO3Auto

AI content factory skeleton for:

- Story generation from ideas and article URLs
- Transcript extraction from YouTube, uploaded video, and audio
- Prompt studio with style/genre presets and character sheets
- Batch video generation with VEO3 queue/progress/retry/autosave foundations
- Batch image generation with Whisk/Nano Banana foundations
- Character continuity workflows with start/end frame linking

## Monorepo layout

- `apps/web`
- `apps/api`
- `packages/domain`
- `packages/application`
- `packages/infrastructure`
- `workers`
- `docs`

## Quick start

```bash
npm install
npm run typecheck
npm run build
```
